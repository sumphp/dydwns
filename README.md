# **김용준의 모바일 앱**

YJ/app      <- 만든 앱

ㄴ      🧢 Cloth-Up (클로즈업)
        "Close up your Clothes!" — Firebase 기반 실시간 패션 SNS 안드로이드 앱
        📖 프로젝트 개요 (Overview)
        Cloth-Up은 사용자가 데일리 룩(OOTD)을 기록하고 공유하는 SNS 플랫폼이다.
        기획 -> 서버환경 만들기 -> 앱 인터페이스 만들기 -> 핵심기능 구현 -> 짜잘한 기능 최적화
        
        -------------------------------------------------------------------
        서버환경
        Firebase 프로젝트를 안드로이드 스튜디오와 연동 후 데이터베이스와 스토리지 사용.
        스냅샷을 활용하여 새로고침이 필요없이 실시간으로 데이터를 동기화할 수 있다.
        
        -------------------------------------------------------------------
        핵심기능
        ActivityResultLauncher를 이용한 갤러리 이미지 접근 권한 획득
        스토리지 이미지 업로드 및 Download URL 추출 로직 구현
        이미지 업로드 비동기처리 addOnSuccessListener를 사용하여 파일이 전송되는 동안 UI 작동
        
        -------------------------------------------------------------------
        문제1. 파이어베이스 스토리지 사용하려면 요금제 업그레이드를 요구해서 카드등록을 해야만 했음.
        문제2. 회원가입 시스템이 없다.
        문제3. 게시물 삭제기능, 댓글기능이 없다.
        
        ** 배운점 Ui 이외로 파이어베이스와 연동을 해야하는 기능들의 코드는 ai의 도움을 받았다.
                ai의 주석설명 덕분에 많이 배웠다.
