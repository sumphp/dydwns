# **김용준의 모바일 앱**

YJ/app      <- 만든 앱

🧢 Cloth-Up (클로즈업) <br />
        "Close up your Clothes!" — Firebase 기반 실시간 패션 SNS 안드로이드 앱
        <br />
📖 프로젝트 개요 <br />
        Cloth-Up은 사용자가 데일리 룩(OOTD)을 기록하고 공유하는 SNS 플랫폼이다.<br />
        기획 -> 서버환경 만들기 -> 앱 인터페이스 만들기 -> 핵심기능 구현 -> 짜잘한 기능 최적화
        
        -------------------------------------------------------------------
        서버환경
        Firebase 프로젝트를 안드로이드 스튜디오와 연동 후 데이터베이스와 스토리지 사용.
        스냅샷을 활용하여 새로고침이 필요없이 실시간으로 데이터를 동기화할 수 있다.
        
        -------------------------------------------------------------------
        핵심기능
        ActivityResultLauncher를 이용한 갤러리 이미지 접근 권한 획득
        스토리지 이미지 업로드 및 Download URL 추출 로직 구현
        이미지 업로드 비동기처리 addOnSuccessListener를 사용하여 파일이 전송되는 동안 UI 작동
        
        -------------------------------------------------------------------
        문제1. 파이어베이스 스토리지 사용하려면 요금제 업그레이드를 요구해서 카드등록을 해야만 했음.
        문제2. 회원가입 시스템이 없다.
        문제3. 게시물 삭제기능, 댓글기능이 없다.
        
        ** 배운점 Ui 이외로 파이어베이스와 연동을 해야하는 기능들의 코드는 ai의 도움을 받았다.
                ai의 주석설명 덕분에 많이 배웠다.

화면별 기능 및 구현사항.

 1. 메인 피드 (Home Screen)

        모든 사용자가 올린 최신 코디를 한눈에 볼 수 있는 앱의 메인 화면.

        핵심 기능: 실시간성

        구현 상세:

        Firebase Firestore의 스냅샷을 연동하여, 별도의 새로고침 없이도 새로운 게시물이 올라오는 즉시 피드 상단에 자연스럽게 추가됨.

        LazyColumn을 사용하여 최적화.

2. 검색 및 탐색 (Search Screen)

        관심 있는 스타일이나 아이템을 찾고, 새로운 트렌드를 발견하는 화면.

        핵심 가치: 언어 필터링

        구현 상세:

        사이드 필터링: 검색어를 입력할 때마다 서버 통신 없이 앱 내부 메모리에서 데이터를 필터링하여, 지연 시간 없는 즉각적인 검색 결과를 나타냄.

        추천 태그: 검색어가 없을 때는 #고프코어, #Y2K 등 인기 스타일 태그를 칩 형태로 제공하여 사용자의 탐색을 도움.
   
3. 코디 업로드 (Upload Screen)

        나만의 데일리 룩을 사진과 함께 기록하고 공유하는 공간.

        핵심 가치: 빠른 업로드, 비동기 처리 

        구현 상세:

        안드로이드의 ActivityResultLauncher를 활용하여 갤러리 접근 후 사진을 선택가능.

        업로드 버튼 클릭 시, 이미지 파일과 게시글 데이터를 비동기적으로 연결하여 데이터를 안전하게 저장.

4. 프로필 (Profile Screen)

        나의 활동을 모아보고 관리하는 개인화된 공간.

        핵심 가치: 나의 프로필 카드, 계정 필터링

        구현 상세:

        whereEqualTo: Firestore에서 데이터를 가져올 때 닉네임이 일치하는 게시물만 가져와 데이터를 띄움.
